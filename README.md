# react-band
A simple react scaffold with webpack, babel, jest, enzyme and puppeteer.

# Document

[en](./en.md)

[zh-CN](./zh-CN.md)

# Install
npm install

# Lint
npm run lint

npm run stylelint

# Unit test
npm run test

npm run test:update

npm run test:coverage

# Run
npm run start

npm run start:mock

visit http://localhost:3000

# Build
npm run build
